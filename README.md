# Diabetes Prediction using Machine Learning

## Overview

This Machine Learning project predicts whether a person has diabetes or not using Logistic Regression.

The model is trained using the PIMA Indians Diabetes Dataset.

---

## Features
- Data preprocessing
- Train-test split
- Logistic Regression model
- Accuracy evaluation
- Prediction system

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab
## Dataset

Dataset used:
PIMA Indians Diabetes Dataset

Features include:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
